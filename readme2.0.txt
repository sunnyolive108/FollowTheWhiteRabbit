Um den Hasen anständig zu jagen, holt man es sich folgende dependencies:

PyQt5: für die UI
# pip install PyQt5

pika: für RabbitMQ
# pip install pika

Du benötigst noch die img/rabbitmq.png im selben Ordner wie die .py
ohne den, wird's nich geh'n


Eingabe:

Username und Password: Deine credentials bekommst Du von deinen Admins
Hostname: Den hostnamen oder die IP Adresse des RabbitMQ Servers (findest Du unter 'Aktuelle Installationen')
Virtual Host: Sofern nicht anders gewollt, ist diese Applikation ausschließlich des Virtual Host's 'BorC3'

Deinen eigenen RabbitMQ Server erreichst Du unter den Default credentials 'guest' und 'guest' auf 'localhost'
Den TEST RabbitMQ Server gibt es auch! 


----------
Schmeiß das Programm!

python jsonthewhiterabbit.py
